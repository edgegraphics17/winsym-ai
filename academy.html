Set Permissions — The Most Important Step

Right after connecting, you'll see a permissions list. These decide what Claude can do on its own.

What Claude Can DoSettingWhy
View generations✅ Always AllowSafe — read-only
Explore models✅ Always AllowSafe — read-only
Generate images⚠️ Ask PermissionCosts credits
Generate videos⚠️ Ask PermissionCosts more credits
⚠ Why this matters: There's no credit counter visible inside Claude. If Claude has full permission, it can fire off dozens of expensive generations in one chat — and you won't notice until credits are gone. This is a real, documented mistake.

Rule: Read-only = Always Allow  ·  Generating = Always Ask.

Create Videos Properly

Seedance 2.0 is the most powerful video model in Higgsfield. How you use it is the difference between mediocre and professional results.

Seedance 2.0 Capabilities
15s
Max per generation
1080p
Max resolution
3000
Max prompt chars
Seedance can create multiple camera angles and cuts from a single prompt, combine real footage with AI animations, and generate transitions between two exact images.
The Beginner Mistake
✗ Short Prompt

"A headphone floating in a room."

→ Lifeless, mediocre result. No movement, no atmosphere.

✓ Detailed Prompt

"0–3s: Camera slowly pushes in, soft bokeh, warm studio light. 3–6s: Slow 360° rotation, metal reflections visible..."

→ Professional result.

Solution: Always let Claude write the prompts using the Seedance Skill. Never write Seedance prompts yourself.
The Seedance Workflow
Step 1 — Ask Claude to write the prompt
Write me a Seedance 2.0 video prompt.
[Describe what should happen]
Duration: [e.g. 10 seconds]
Include detailed camera movements, lighting, and atmosphere.
Step 2 — If Claude says the prompt is too long
Make this prompt under 3000 characters, keep all key details.
📌 Steps 3–4: In Higgsfield
3.Copy everything starting from "Shot 1:" — skip Master Effects and Effects Density sections
4.Navigate to Seedance 2.0 → upload start frame (optional) → paste prompt → set duration → Generate
Multi-Shot vs. Single-Shot
✗ Single-Shot

Generate each 2-second scene individually → inconsistent results, much more effort, worse quality.

✓ Multi-Shot

Claude writes one 15-second prompt with multiple scenes → Seedance handles cuts and transitions automatically.

Multi-shot template
Write a Seedance prompt for a 15-second video with 3–4 different shots/cuts.
Seedance should handle the transitions. [Describe the content]
Combining Real Footage with AI

Film yourself → AI picks up exactly where you left off → seamless transition. Works for portals, explosions, transformations, VFX overlays.

📌 Workflow
1.Record your video on your phone
2.Export the exact frame where the AI effect begins as a still image
3.Upload the still to Claude with the prompt below
4.Upload the still as Start Frame in Seedance → paste prompt → Generate
5.Place the AI video directly after your real footage in your editor
Prompt for real footage transition
Write a Seedance prompt where [DESCRIBE EFFECT] happens.
Make sure the start frame is IDENTICAL to the reference image provided.
Same lighting, same character position, same environment.
Music — Important Limitation
⚠ Problem: Seedance automatically adds music. With multiple scenes, the music won't be consistent between them.

Solution: Always specify "No music in the video. Sound design and ambient sounds only." — add music separately in CapCut or DaVinci Resolve.
Troubleshooting
06

Handle Errors

Content blocks and style drift are the two most common problems. Neither is permanent — here's exactly how to fix each one.

Content Blocks

Sometimes Higgsfield rejects a generation. For images, credits are refunded. For videos, you may see "Rights Verification Required."

📌 Fix: Rights Verification Required
1.Open higgsfield.ai directly (not Claude)
2.Go to Video Generations
3.Find the entry with that status → click through and confirm
4.Back to Claude → reload the conversation → video now appears
Fix: Rejected Prompt
This generation was blocked. Please analyze the prompt, identify
which words or phrases likely caused the issue, rewrite the prompt
without those elements and try again.
Style Consistency Lost
Problem: After 3–4 scenes, images suddenly look different from the first ones.
Cause: Claude continued without an explicit style reference.
Fix: Restore Style Consistency
The style changed. Use image 2 as the visual style reference
for all remaining generations. Regenerate the ones that don't match.

When you've produced a result you want to reproduce consistently — turn it into a skill. A skill is the recipe for Claude: without it, results vary every time.

The 5-Step Process
📌 How to Build a Skill
1.Find your best result — look through all generated assets and pick your favorite
2.Copy the exact prompt from your tracking sheet
3.Let Claude write the skill — use the prompt below
4.Test it — open a new chat, call the skill, generate an asset, verify the style matches
5.Improve it — give Claude feedback on what you liked and didn't like
Skill Creation Prompts
Step 3 — Create the Skill
This prompt above is my best result from Higgsfield Marketing Studio.
It was a Hypermotion-style product video – fast cuts, professional details.

Turn this into a skill and save it under .claude/skills/hypermotion-video.md

The skill should:
- Always reproduce this style
- Ask 1–2 clarifying questions before starting (model in video? UGC or product-only?)
- Avoid phrases that have caused content blocks in the past
Step 5 — Improve the Skill
You just generated 5 ads with this skill.
I like A, B, C. I don't like D, E.
Update the skill so future generations are more like A, B, C.

Build a Tracking System

Without tracking you don't know which prompts worked, which videos were already generated, or what separates your best ads from your worst. The Status column is everything.

Create Your Google Sheet

and Status column (Planned / In Progress / Done / Review)

Set Up Automation

Routines are scheduled prompts. Claude executes them automatically — without you being present. Set them up once and your pipeline runs itself every week.

The Weekly Cycle
Sunday 8 PM — Planning Routine (Set Up Once)
Every Sunday at 8 PM:
1. Open the Google Sheet and analyze all past generations
2. Read the advertising knowledge file
3. Create 50 new generation ideas for the week
4. Add them to the sheet with Status "Planned"
Monday 6 AM — Production Routine (Set Up Once)
Every Monday at 6 AM:
1. Open the Google Sheet
2. Select 30 entries with Status "Planned" (highest priority first)
3. Build the prompts using the skill files
4. Generate all 30 via Higgsfield
5. Mark each as "Done" when the URL is returned
6. Save all Job IDs and result URLs in the sheet

Critical Mistakes
11

What NOT To Do

These mistakes come directly from documented real-world practice. Every single one has cost someone real money or real time.

The 7 Critical Mistakes
01
Set All Permissions to "Always Allow"
Credits disappear without you noticing. Fix: Keep generation permissions on "Ask Permission" always.
02
Not Providing a Reference Image
Your product looks different in every video. Fix: Always include the reference image for every project.
03
Writing Short Prompts for Seedance
Lifeless, generic videos every time. Fix: Always let Claude write the prompts using the Seedance Skill.
04
Single-Shots Instead of Multi-Shot
More work, worse results, inconsistent style. Fix: Write one 15-second multi-shot prompt instead.
05
Leaving Music On in Multi-Scene Videos
Music is inconsistent between scenes. Fix: Always specify "no music," add it separately in post.
06
No Tracking System
Duplicate generations waste credits. Fix: Always maintain the Status column in your tracking sheet.
07
Never Using Higgsfield Directly First
You don't understand credit costs and make expensive mistakes. Fix: Spend 1–2 weeks in Higgsfield directly before adding Claude.
