## PART 5: VERCEL → GOOGLE RANKING (Full Roadmap)

### Stage 1: Technical Foundation (Day 1-3)
```
1. DOMAIN SETUP
   - Connect winsym.ai to Vercel (DNS → A record or CNAME)
   - Force HTTPS (Vercel auto-handles)
   - Redirect www → non-www (or vice versa) — pick ONE, stick to it

2. SITEMAP (sitemap.xml)
   <?xml version="1.0"?>
   <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
     <url><loc>https://winsym.ai/</loc><priority>1.0</priority></url>
     <url><loc>https://winsym.ai/projects</loc><priority>0.8</priority></url>
     <url><loc>https://winsym.ai/academy</loc><priority>0.8</priority></url>
     <url><loc>https://winsym.ai/about</loc><priority>0.6</priority></url>
   </urlset>

3. ROBOTS.TXT
   User-agent: *
   Allow: /
   Sitemap: https://winsym.ai/sitemap.xml

4. META TAGS (Per Page)
   Homepage:
   - Title: "AI Coach & Consultant Kuala Lumpur | winsym.ai"
   - Description: "Karim Azzaoui builds AI systems for businesses in Malaysia.
     Company Brain method. Google Partner. Book a free discovery call."
   - OG tags for social sharing
   
5. STRUCTURED DATA (JSON-LD in <head>)
   - LocalBusiness schema (KL location, phone, hours)
   - Person schema (Karim Azzaoui, founder)
   - Service schema (each service offering)
   - FAQPage schema (common objections as FAQ)

6. CANONICAL URLS
   - Each page has <link rel="canonical" href="https://winsym.ai/[page]">
   - No duplicate content
```
